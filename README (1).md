## ✨ Features

- 🌙 **Dark / Light Mode Toggle** (persisted in localStorage)
- 🖼️ **Lightbox Gallery** with keyboard navigation (←→ Esc)
- 🎛️ **Gallery Filter** by category (Events / Certificates / Projects)
- 📑 **Tabbed Achievements** section
- 🎞️ **Scroll Reveal Animations** using IntersectionObserver
- 📱 **Fully Responsive** — works on all screen sizes
- ⚡ **Zero dependencies** — pure HTML/CSS/JS, no frameworks

## 🚀 How to Deploy on GitHub Pages

1. Create a new GitHub repository (e.g., `portfolio`)
2. Upload all files maintaining the folder structure above
3. Go to **Settings → Pages**
4. Set Source to `main` branch, root `/`
5. Your site will be live at `https://yourusername.github.io/portfolio`

## 🖼️ Replacing Placeholder Images

All images in `assets/images/` are currently placeholder SVGs.  
Replace each file with your actual photo/screenshot/certificate (`.jpg` or `.png`) keeping the **same filename**.
